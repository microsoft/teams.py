# Security Practices

All Northwind Co. employees are responsible for protecting company and customer data. This policy
applies to all devices, accounts, and networks used for company work.

## Account Security

- Enable multi-factor authentication (MFA) on every company account within 24 hours of receiving
  credentials. MFA via hardware key is preferred; TOTP apps are acceptable; SMS is not.
- Passwords must be at least 14 characters and stored in the company-approved password manager.
- Never share credentials, even with IT. Legitimate support requests will never ask for a password.

## Device Security

Company laptops must run the latest OS security patches within 7 days of release. Disk encryption
and the endpoint agent are mandatory and non-removable. Do not install unapproved software; check
the internal software catalog or request approval through IT.

## Data Handling

Customer data must remain in approved systems. Do not export customer records to personal cloud
storage (Dropbox, personal Google Drive, etc.) under any circumstances. Local copies on your laptop
are permitted for active work but must be deleted when the work is complete.

## Reporting Incidents

If you suspect a security incident — phishing, lost device, unauthorized access — report it
immediately to `security@northwind.example` and call the security hotline at x5911. Early reporting
is rewarded; there are no penalties for reporting a mistake promptly.
