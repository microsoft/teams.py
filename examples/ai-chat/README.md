# ai-chat

An AI-powered Teams bot that streams responses in real time and remembers your conversation history.

Built with [PydanticAI](https://ai.pydantic.dev) and the Microsoft Teams Python SDK.

## What this sample demonstrates

| Teams feature | Where to look |
|---|---|
| **Streaming** | `ctx.stream.emit(chunk)` — token chunks from PydanticAI piped live into Teams |
| **"Generated by AI" label** | `MessageActivityInput().add_ai_generated()` on the final streamed message |
| **Memory** | `message_history=` on each `agent.run_stream()` call, stored per conversation |

## Why PydanticAI

The Teams SDK is AI-framework agnostic. `microsoft-teams-apps` and `microsoft-teams-api` provide the Teams integration layer (streaming, activity routing, the AI label). The model call is entirely up to you.

PydanticAI is used here because:
- **Model-agnostic**: one string change to switch providers (see below)
- **Clean streaming API**: `result.stream_text(delta=True)` yields token deltas directly
- **Native history format**: `result.all_messages()` returns the full conversation as `list[ModelMessage]`, ready to pass back as `message_history=` on the next turn

## Setup

### Prerequisites

- Python >= 3.12
- [UV](https://docs.astral.sh/uv/) >= 0.8.11
- An OpenAI API key (or another supported provider — see below)

### Environment variables

Create a `.env` file in this directory:

```
OPENAI_API_KEY=your-key-here
OPENAI_MODEL=gpt-4o        # optional, defaults to gpt-4o
PORT=3978                  # optional, defaults to 3978
```

### Run

```bash
uv sync
uv run src/main.py
```

Then use [Teams Toolkit](https://aka.ms/teams-toolkit) or [Dev Proxy](https://aka.ms/dev-proxy) to connect a Teams client to `http://localhost:3978`.

## Switching models

Change the `_model` assignment in `src/main.py`. No other code changes are needed:

```python
# OpenAI (default)
_model = f"openai:{getenv('OPENAI_MODEL', 'gpt-4o')}"

# Anthropic
_model = "anthropic:claude-3-5-sonnet-latest"

# Google
_model = "google-gla:gemini-2.0-flash"

# Groq
_model = "groq:llama-3.3-70b-versatile"
```

Each provider requires its own API key environment variable. See the [PydanticAI models documentation](https://ai.pydantic.dev/models/) for the full list.

## Commands

| Message | Effect |
|---|---|
| Any text | Streamed AI response with conversation history |
| `clear` | Reset this conversation's history |
