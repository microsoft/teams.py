## Installation

```bash
uv add microsoft-teams-devtools
```

## Usage

```python
from microsoft.teams.apps import App
from microsoft.teams.devtools import DevToolsPlugin

app = App()
app.use(DevToolsPlugin(port=3979))

await app.start()
# Open http://localhost:3979/devtools in your browser
```
